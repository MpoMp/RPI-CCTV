Home Security Camera; Issues

  • Cost; camera, cables, storage, setup.
  • Complexity; house modifications, camera setup, storage and recording.
  • Access; local or remote, access control, protection.
  • Ownership; yours or by a third-party.

Common solutions for home CCTV

  • Cost range from 400 to 3000+ euro...
  • Difficulty in installing and configuring.
  • In some cases there are no options for customization at all.
  • You have to use specific types of storage and cables.

The RPI solution

  • Cheap and scalable; use a simple USB camera or an HD security camera.
  • Easy to setup; hundreds of guides for setting up Raspbian and the camera software.
  • Highly customisable; based mostly on open software and flexible hardware.
  • Accessible and secure; access locally or via the Internet, use and configure any type of security measures you prefer.

Topology

How it works (Components)

  • Raspberry Pi; low cost, credit-card sized computer
  • Simple camera; RPI camera module or any regular USB camera
  • Router; your home router and a regular DSL connection
  • Storage; use a cloud storage service, your own server or local storage (even RPI's SD card!)

How it works (Software)

  • Raspbian; free operating system based on Debian, optimized for the Raspberry Pi hardware
  • Motion (or Motion MMAL); highly customizable software motion detector, written in C for Linux
  • Motion Notify; notification system for Motion providing upload to Google Drive and email notification, written in Python
  • Remote server setup; (optional) Motion provides a simple video stream but you can customize your remote access interface

How it works (Software)

  • You can use any other plugin for motion detection and notifications.
  • Even write your own and use another cloud service (OneDrive, Dropbox, Mega, etc.)!
  • You may use other notification software that sends messages to you on your mobile or laptop anywhere (e.g. Growl).
  • Or setup an email notification system of your own and upload to a server you own via FTP.
  • Example of uploads: Dropbox, Drive
